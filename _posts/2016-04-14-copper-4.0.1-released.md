We are proud to anounce, that COPPER 4.0.1 has just been released.
This release mainly contains some minor enhancements:
- New feature: Some extensions in the JMX interface
- New feature: In persistent engines using an RDBMS, responses provided to getAndRemoveRespone(s) are ordered by response TS

You can get it here:
[https://github.com/copper-engine/copper-engine/releases/tag/4.0.1](https://github.com/copper-engine/copper-engine/releases/tag/4.0.1)
