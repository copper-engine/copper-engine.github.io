COPPER 6.0.0 has been released.
It is available on [GitHub](https://github.com/copper-engine/copper-engine) and in [Maven Central](https://search.maven.org/search?q=g:org.copper-engine%20AND%20a:copper-coreengine&core=gav).

COPPER has been upgraded to work with Java 17-25. Older versions of Java are supported in previous COPPER versions.
COPPER 5.4 runs with Java 8 to 18. Java 7 is supported by COPPER 4.x.
