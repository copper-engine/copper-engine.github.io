COPPER 7.0.0 has been released.
It is available on [GitHub](https://github.com/copper-engine/copper-engine/releases) and in [Maven Central](https://central.sonatype.com/artifact/org.copper-engine/copper-coreengine/versions).

Added the optional usage of virtual threads. COPPER 7 itself will be using Java 21.
