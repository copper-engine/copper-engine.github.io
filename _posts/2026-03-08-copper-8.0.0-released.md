COPPER 8.0.0 has been released.
It is available on [GitHub](https://github.com/copper-engine/copper-engine/releases) and
in [Maven Central](https://central.sonatype.com/artifact/org.copper-engine/copper-coreengine/versions).

Virtual threads in Processor Delegates now will run on Java 25. 
COPPER 8 itself will be using Java 25.
