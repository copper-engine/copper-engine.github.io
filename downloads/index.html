COPPER is on Maven Central, too. You may use the following to include the COPPER core workflow engine in your Maven project: {% highlight xml %} org.copper-engine copper-coreengine 3.0 {% endhighlight %} If you are using Gradle, simply add the following to your dependencies section: {% highlight groovy %} compile 'org.copper-engine:copper-coreengine:3.0' {% endhighlight %} If you are using COPPER with Spring you also need to include copper-spring: {% highlight xml %} org.copper-engine copper-spring 3.0 {% endhighlight %} and likewise for Gradle: {% highlight groovy %} compile 'org.copper-engine:copper-spring:3.0' {% endhighlight %}
